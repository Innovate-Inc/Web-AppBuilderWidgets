Splash

Splash widget defines the display content on the app's splash screen. It appears before users start to interact with the app.

The following table shows the descriptions of the configurable attributes.

Attribute

Description

*splash

Object; default: no default --It has following properties .

  • *splashContent: String; default: no default —The content displays in Splash widget.
  • *requireConfirm: Boolean; default: false; If true, it requires confirmation to continue.
  • *confirmText: String; default: Do not show this splash screen again; It is the label for the confirmation checkbox.

Example:

{
  "splash": {
    "splashContent": "<div style=\"text-align: center;\"><a href=\"http://www.esri.com/\" target=\"_self\">Hello world!</a><br /></div>",
    "requireConfirm": false,
    "confirmText": "I agree to the above terms and conditions"
  }
}
