Analysis widget

The Analysis widget provides an easy way to use ArcGIS Online spatial analysis tools in your application. With 22 analysis tools, the same as those in ArcGIS Online map viewer, you can configure one tool per widget or multiple tools in a widget. Refer to the Perform analysis section of ArcGIS Online help for the introduction of spatial analysis, the license requirements to access analysis tools, and details about the usage of each tool. If you're a developer, you can access these tools via web APIs. For more information, see Spatial analysis service REST API.

Using spatial analysis service consumes credits. For more information on credits, see Service credits overview that includes access to an interactive Service credits estimator.

Caution:

You cannot run any of the analysis tools using data from ArcGIS Server map or feature services that are not publically accessible. That is, the URL to the service must be a public URL, not one only accessible behind a firewall. If you use a non-public map or feature service, you'll receive an error saying that the URL is not accessible.

Configuring the Analysis widget

Steps:
  1. Hover over the Analysis widget and click the small edit icon . The configuration window for this widget opens.
  2. 22 tools alphabetically list in the configuration window. Choose the tools that you want to include in the widget.
  3. Click OK to save the configuration and close the window.

Using the Analysis widget

License:

To perform analysis, the administrator of your organization needs to grant you certain privileges. To use any of the analysis tools, you need the following privileges:

  • Create, update, and delete content
  • Publish hosted features
  • Spatial analysis

Certain tools need additional privileges such as network analysis and GeoEnrichment. See the Perform analysis section of ArcGIS Online help for more information. If you don't have required privileges, the corresponding tools will not be available in the widget panel.

Steps:
  1. Verify that you are signed in to your organization with privileges to perform analysis.
  2. Click the widget icon in your application to open the widget panel. If more than one tool is configured, choose a tool one at a time to execute. If only one tool is available, it activates automatically. Click the help icon for information about the tool.
    3. Note:

    If you don't have required privileges to run a tool, it will not display in the widget panel. Besides privileges, each tool has its own requirement for the layer or layers. If a tool does not have a required layer or layers in the map, it will grey out and no longer be executable.

    Note:

    The Extract Data tool requires extract capability set in the service.

  3. Choose an input layer from the drop-down list and click Next. In this case, Create Buffers tool is chosen to demonstrate the workflow.
  4. Configure parameters for the tool and click Run Analysis.
    5. Tip:

    See the Work with a tool dialog box section of ArcGIS Online help for more information about the configuration dialog box.

    Optionally, if you want to know the credit usage of the tool, click Show credits.
  5. When the tool is running, the execution messages display in the panel. Click the cancel icon if you want to cancel the job.

    A job for a tool consuming credits cannot be cancelled.

    When the job is successful, the result is saved to My Content. Click the item link under Outputs to view the item details in ArcGIS Online. In addition, the result as feature layer or layers, or table or tables are automatically added on the map as operational layers.

    When the job is not successful, an error message displays in red. Click Back to return to the parameters' configuration.

  6. If you have more than one tool, click Home to choose other tools.
