Widget on screen configuration

The following table shows the attributes and descriptions.

Attribute

Description

panel

object. Default—BaseWidgetPanel. Should have URI property. All widgets (not in group) display in this panel.

"panel": {
      "uri": "jimu/PanelType1"
    },

widgets

An array of widgets. Each widget has two additional properties:

  • position—Same as the map's position.
  • positionRelativeTo—string. The default is map. Valid value is map or browser.

groups

Array of groups.

Example:

"widgetOnScreen": {
    "panel": {
      "uri": "jimu/PanelType1"
    },

    "widgets": [{
      "uri": "widgets/Header/Widget",
      "label": "",
      "positionRelativeTo": "map",
      "position": {
        "left": 0,
        "top": 0,
        "right": 0,
        "bottom": 30,
        "width": 100,
        "height": 100
      },

      "openAtStart": true,
      "config": {}
    }],

    "groups": [{
      "panel": {
        "uri": "jimu/PanelType1",
        "positionRelativeTo": "map",
        "position": {
          "left": 0,
          "top": 0,
          "right": 0,
          "bottom": 30,
          "width": 100,
          "height": 100
        }
      },

      "widgets": [{
        "uri": "widgets/Header/Widget",
        "label": ""
      }]
    }]
  },
