Swipe

The Swipe widget provides a simple tool to show a portion of a layer or layers on top of a map.

The following table shows the descriptions of the configurable attributes.

Attribute

Description

style

String. Type of swipe tool to use. Default value is vertical. Available options are vertical, horizontal and scope.

layerId

String. The ID of the layer to swipe. No default value.

Example:

{
    "style": "vertical",
    "layerId": "SuperTuesdaySample_1433"
}
