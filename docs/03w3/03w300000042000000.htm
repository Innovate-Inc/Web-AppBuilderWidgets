Open multiple widgets simultaneously

For in-panel widgets in a widget pool, the widget behavior is controlled by the Controller widget, such as HeaderController and PanelManager.

The HeaderController widget closes other widgets/groups when a widget/group is opened. It also can open all widgets in the same group at the same time in one panel.

The PanelManager closes other widgets in the same group when a widget is opened.

As a result, if you want to open more than one widget at the same time, here are the options:

  • Customize the HeaderController widget or create a new controller widget that will not close other widgets/groups when a widget/group is opened.
  • Put some widgets in the widgetOnScreen section and some in the widgetPool section.
  • Put the widgets in the widgetOnScreen's group and configure a panel for the group. All groups in the widgetOnScreen section open when the app loads.
  • Make the widget off-panel and configure it in the widgetOnScreen section. All off-panel widgets in the widgetOnScreen section open when the app loads.
