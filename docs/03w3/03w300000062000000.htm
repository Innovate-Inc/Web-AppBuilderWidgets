What's new in version 1.1

Web AppBuilder for ArcGIS (Developer Edition) 1.1 was released in March 2015. It includes the following features and enhancements:

  • New Time Slider, Swipe and Analysis widgets.
  • Support for related records in pop-up, attribute table and query results.
  • Support for feature search based on search configuration in web map.
  • Support for customizing map's visible scales.
  • Support for using URL parameters for the app.
  • Support for importing existing apps from Web AppBuilder (Developer Edition), Web AppBuilder integrated in ArcGIS Online, or Web AppBuilder integrated in Portal for ArcGIS.
  • Support for upgrading apps created in the previous version of Web AppBuilder to a newer version.
  • Enhancement to Layer List with enable/disable pop-up option and new type support including table, raster and image service layers.
  • Enhancement to Overview Map, Draw and Geoprocessing widgets.
  • Support for Greek.
